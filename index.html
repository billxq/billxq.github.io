excel计算年龄

近期在工作中经常遇到通过一个身份证来计算年龄的问题，于是深入研究了一下excel的几个关于日期的函数，year(), date(), today(), mid(), datedif()。现总结如下：

[toc]

初步了解一下以上几个函数

year()函数

返回对应某个日期的年份

例如：year(“2021/1/22”) 返回2021

date()函数

date函数返回表示特定日期的连续序列号

例如： date(“2021”, “1”, “22”) 返回： ”2021/1/22“

today()函数

返回今天的日期， 形如： ”2021/3/25“

mid()函数

返回文本字符串中从指定位置开始的特定数目的字符，该数目由用户指定。

syntax:

  • MID(text, start, num_chars)

    • text 文本内容，必需，如身份证号码：310229198502260820

    • start 需要提取的起始位置，出生年份是第7个

    • num_chars 需要提取多少个字符，从年份到日期共8位

      因此从身份证提取出生年月就可以写成： =MID("310229198502260820", 7, 8)

datedif()函数

计算两个日期之间相隔的天数，月数或年数。

syntax:

​ DATEDIF(start_date, end_date, unit)

        - start_date: 给定一个其实日期，例如："2021/1/30"
        - end_date: 给定一个结束日期，例如： ”2023/2/11“
  • unit: 指定需要返回的信息类型
            - Y： 返回整年数
             - M： 返回整月数
             - D:  返回一段时期内的天数，比较常用

docker网络

none网络

挂在这个网络的容器，除了lo，没有其它任何网卡。可以通过–network=none来指定使用none网络。使用场景，如安全性相对较高的应用，可以使用none网络。可以让这个容器随机生成密码，就可以放到none网络，避免密码被窃取。

host网络

容器的网络配置与host完全一致。通过–network=host来指定容器使用host网络。对网络传输要求较高的容器可以选择host网络，但是要考虑端口冲突问题。

bridge网络

这是docker默认的网络方案，创建的容器默认会挂载到docker0上。可以通过docker network inspect bridge命令来查看bridge网络的配置信息。

user-defined网络

Docker提供三种user-defined网络驱动：bridge，overlay和macvlan。overlay和macvlan用于创建跨主机的网络。创建方法：docker network create --driver bridge --subnet 172.22.16.0/24 --gateway 172.22.16.1 my_net
